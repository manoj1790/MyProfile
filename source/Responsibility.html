Responsibilities

Profile > Responsiblities

Developer / Team Lead:
  • Manage team activities from planning, coordinate and deliver project deliverables on time with quality.
  • Represent team in project discussions and meetings.
  • Prepare Project Design document and work estimates.
  • Code and test modules both batch and online.
  • Write unit test scripts. Perform Unit and regression testing for the changes.
  • Monitor batch application defects during SIT / UAT testing.
  • Fix issues in the test environments.
  • Request for DB changes and run adhoc process to update databases records.
  • Provide support in SIT / UAT environment, testing cycles, project implementation and Post implementation support during warranty period.
  • Maintaining the application and opportunities for improving system health and enhance of the application.
  • Document Rewards process for future references.
System Analyst:
  • Gather project requirements from business and functional requirement.
  • Prepare high level design document and review with all stake holders.
  • Respond to business user queries and integration issues.
  • Coordinate UAT testing with business users.
  • Review project documents and application defects in every release.
  • Analyse the application defects, prepare RCA and propose fix.
Support Lead:
  • Transition the support related activities from incumbent support team / application teams.
  • Review all support related documents.
  • Create the process flow and educate users on the support structure and process.
  • Manage the team support activities, SLAs and address all escalations.
  • Manage infra support and continuity of business.
  • Monitor application heath check and infrastructure maintenance.
  • Onboarded automation tools like AppDynamics, auto watch plus and One Touch Cob (OTC).
